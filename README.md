# 📌 Overview

Real-Time Text-to-Image Generator is a Generative AI application that converts natural language prompts into visually compelling images in real time.

The project demonstrates how modern diffusion-based image generation models can transform textual descriptions into high-quality visual content through an intuitive interface.

This project explores practical applications of **Generative AI**, **Prompt Engineering**, and **Text-to-Image Synthesis**.

- 🎨 Real-time text-to-image generation

# 🎯 Problem Statement

Creating custom digital artwork traditionally requires artistic expertise and significant time.

This project simplifies content creation by enabling users to generate images directly from natural language descriptions using Generative AI, making creative design more accessible and efficient.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/AdityaPatil7900/Real-Time-Text-To-Image-Generator---Gen-AI.git
```

Navigate to the project

```bash
cd Real-Time-Text-To-Image-Generator---Gen-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
